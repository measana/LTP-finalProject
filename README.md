# Sistema de Cadastro de Computadores 

## Descrição
Este projeto foi desenvolvido em **Java** como trabalho final.  
O objetivo é gerenciar um cadastro de computadores, permitindo operações de:
- Inclusão (registro de novos computadores)
- Alteração (edição de dados já cadastrados)
- Exclusão (remoção de registros)
- Consulta (visualização dos computadores cadastrados)

O sistema oferece uma interface simples e funcional para manipulação dos dados, garantindo organização e facilidade de uso.

---

## Funcionalidades
- **Cadastrar computador**: inserir informações como modelo, fabricante, especificações e descrição.
- **Alterar registro**: atualizar dados de um computador já existente.
- **Excluir registro**: remover um computador do sistema.
- **Consultar registros**: listar todos os computadores cadastrados ou buscar por critérios específicos.

---

## Tecnologias Utilizadas
- **Java SE** (versão 8 ou superior)
- **IDE**: Eclipse / IntelliJ / NetBeans (qualquer compatível)
- **Banco de Dados**: pode ser implementado com arquivos `.txt` ou banco relacional (ex.: MySQL), dependendo da versão escolhida.
